System Requirements
  • Minimum System Requirements for operation of the SRTE:
    • Windows XP or Windows Vista (32-bit)
    • 2 GHz Processor
    • 1 GB of RAM
    • 50 MB of free disk space

  • Recommended System Requirements for operation of the SRTE:
    • Windows XP or Windows Vista (32-bit)
    • 4 GHz Processor
    • 2 GB of RAM
    • 60 MB of free disk space
